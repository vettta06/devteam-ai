# DevTeam AI

Интеллектуальный менеджер задач для разработчиков с возможностью автоматического разбиения задач на подзадачи с помощью ИИ.

## Идея

Пользователь описывает задачу → система генерирует структурированный план выполнения (подзадачи) → пользователь управляет прогрессом.

## Стек технологий

- **Backend**: FastAPI
- **База данных**: PostgreSQL (SQLite в тестах)
- **Аутентификация**: JWT + refresh-токены + email-подтверждение
- **ИИ-интеграция**: LLM через Ollama (`gemma3:1b`)
- **Контейнеризация**: Docker + Docker Compose

## Функционал

### Пользователи
- Регистрация с подтверждением email
- Вход по email/паролю
- Защита всех эндпоинтов через Bearer-токены
- Обновление access-токена через refresh-токен

### Задачи
- Создание, просмотр, редактирование, удаление задач
- Каждая задача принадлежит только своему владельцу
- Возможность разбить задачу на подзадачи через ИИ

### ИИ-модуль
- Эндпоинт `/tasks/{id}/split` отправляет описание задачи в LLM
- Возвращает структурированный список подзадач с описаниями

##  Запуск

### Локально (разработка)
```bash
cd backend
uvicorn app.main:app --reload
```

### В Docker

```bash
docker-compose up --build
```

Приложение будет доступно на http://localhost:8000
Интерактивная документация: http://localhost:8000/docs
